Allow Log parser to extract the maximum number of successive failed build retries from the failure log message. Clear to enforce the configuration value to be always used. The Regular expression to search for will need to have a integer number in its first match group e.g. 
ERROR: (?:NETWORK ERROR|ARTIFACTORY ERROR) RETRY: (\d)

and the log message 
ERROR: NETWORK ERROR RETRY: 2
and 
ERROR: ARTIFACTORY ERROR RETRY: 1
in the build console output would trigger 2 and 1 maximum reruns.
